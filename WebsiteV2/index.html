About Me

A Little Information

I've had an interest in the IT industry specifically development from a young age. Work Experience

Network Communication System LTD (June 2021→Current)

My time with NCS i have worked with software. More specifially software testing. Currently i am working on a new door entry & warden call system. Some features of the software are SIP calling, RTSP camera streaming, Tablet to tablet video streaming, MQTT server based on a raspberry pi & privacy options. When APKS are sent on a daily basis I fully test the software to find any new bugs that have occured. I am then tasked with creating detailed bug reports to send to developers to meet customer requirements. During this time i have also had exposure to Linux and MQTT servers via a raspberry pi. This allows users to send pre-defined messages via local network to all tenants in case of emergency.

QA Consulting (July 2019→March 2020)

During my time at QA Consulting i worked as both a Data Scientist & a Devops Engineer. My first project was to create an application that would take in user input. With the knowledge i gained throughout my training with QA I created a UI that would take specific data about a house. The data would go through a linear model which would create a predicted price for that house. The project used SQL as a database. Which has testing data saved to it. This was then linked and sourced into R studio where the application models were created. Once the application was complete I deploted it using GCP which would allow me to access from anywhere at any time. All files were then stored on Github for accessability and cloned into the GCP instance.

My second project was to set up a web app using CI integration using Microsoft Azure. This involved using Jenkins. With the knowledge i gained throughout my time with QA. Once the application was complete all files were stored on Github. I followed this up by setting up an instance in Azure with Jenkins to set up a CI pipeline. When the github files are updated this instantly pulls the new data updating the website.
